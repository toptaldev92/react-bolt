## Getting Started

```sh
$ git clone https://github.com/leonardomso/starboy-boilerplate
$ cd starboy-boilerplate
$ npm install
$ npm start
```

## Features

- React
- Redux
- React Router
- Webpack
- Babel
- ESLint
- Jest & Enzyme
- Thunk & Logger
- Stylelint
- Postcss
